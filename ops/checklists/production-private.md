# Production-Private Checklist

- [ ] Auth gate implemented and verified.
- [ ] HTTPS/private network path verified.
- [ ] Audit events created for approvals, commands, policy changes, auth events, and restricted artifacts.
- [ ] Rate limiting implemented or explicitly blocked.
- [ ] Backups run and restore drill verified.
- [ ] Docker Compose config validated.
- [ ] No public exposure before all checks pass.
